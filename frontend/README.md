# Front End Acne Detection

Aplikasi **Acne Detection** adalah antarmuka pengguna (frontend) berbasis React.js yang dirancang untuk melakukan deteksi jerawat, menampilkan hasil analisis, serta menampilkan riwayat deteksi pengguna. Aplikasi ini terhubung dengan backend Flask (disimpan di repo terpisah) yang menjalankan model YOLO untuk mendeteksi jerawat pada wajah.

Frontend ini memiliki desain yang modern, responsif, dan user-friendly.
